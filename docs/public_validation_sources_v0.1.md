# 公開網路驗證資料來源候選 v0.1

日期：2026-06-17

## 1. 目的

使用者已明確指出：不能只靠使用者個人事件來撐六壬與奇門的動態盤驗算。

因此本文件先建立公開資料來源候選，用於：

1. 六壬起課／解課回測。
2. 奇門起局／解局回測。
3. 動態運算層事件樣本庫。
4. 四術 MVP（最小可行產品）準度驗證。

---

## 2. 優先來源類型

| 類型 | 中文用途 | 適合術數 |
|---|---|---|
| 地震事件 | 時間、地點、震度、結果明確 | 六壬、奇門、應期回測 |
| 天氣災害 | 颱風、豪雨、警報、災情 | 六壬、奇門、風險判斷 |
| 交通事故 | 事故時間、地點、結果明確 | 六壬、奇門 |
| 公開公告事件 | 政府公告、法院判決、公司公告 | 六壬、奇門、八字×紫微事件驗證 |
| 體育賽事 | 開賽時間、勝負結果明確 | 六壬、奇門短期成敗題 |
| 股價／市場資料 | 開盤、收盤、漲跌結果明確 | 六壬、奇門短期趨勢題，但需風險標記 |

---

## 3. 第一批候選來源

### SRC-PUB-001｜USGS Earthquake Catalog API（美國地質調查局地震目錄 API）

- 官方來源：https://earthquake.usgs.gov/fdsnws/event/1/
- 用途：查詢地震事件。
- 優點：時間、地點、規模、深度可查。
- 適合：六壬／奇門事件回測。
- 注意：時間多為 UTC（世界協調時間），需換算台灣時間 UTC+8。

### SRC-PUB-002｜USGS Real-time GeoJSON Feeds（美國地質調查局即時 GeoJSON 地震資料）

- 官方來源：https://earthquake.usgs.gov/earthquakes/feed/v1.0/geojson.php
- 用途：即時地震事件。
- 優點：適合自動抓資料。
- 適合：動態樣本自動化。

### SRC-PUB-003｜Taiwan CWA Open Data（台灣中央氣象署開放資料）

- 官方來源待查：中央氣象署開放資料平台。
- 用途：台灣地震、天氣、颱風、豪雨資料。
- 優點：最貼近使用者地區。
- 狀態：待重新查證 API（應用程式介面）與授權條款。

### SRC-PUB-004｜Open-Meteo Historical Weather API（Open-Meteo 歷史天氣 API）

- 官方來源待查。
- 用途：歷史天氣回測。
- 適合：奇門環境風險、事件背景校正。
- 狀態：待查官方文件。

### SRC-PUB-005｜Sports API / 官方賽事資料（體育賽事官方資料）

- 來源待定：NBA、MLB、足球聯賽等官方比分頁。
- 用途：短期成敗題。
- 優點：結果明確。
- 注意：不可使用賭博網站作主要來源。

## 4. 樣本卡格式

每筆公開事件樣本應固定為：

| 欄位 | 中文說明 |
|---|---|
| sample_id | 樣本編號 |
| event_type | 事件類型 |
| event_title | 事件標題 |
| event_time_utc | 事件 UTC（世界協調時間） |
| event_time_taipei | 事件台灣時間 |
| location | 地點 |
| source_url | 來源網址 |
| source_type | 來源類型 |
| known_result | 已知結果 |
| usable_for | 可用術數 |
| risk_level | 風險等級 |
| notes | 備註 |

---

## 5. Gate（檢核閘門）

| Gate（檢核閘門） | 判定 |
|---|---|
| 沒有事件時間 | BLOCK（阻擋） |
| 沒有地點 | WARN（警告）或 BLOCK（阻擋） |
| 沒有公開來源 | BLOCK（阻擋） |
| 來源不是官方或可靠媒體 | WARN（警告） |
| 結果不可驗證 | BLOCK（阻擋） |
| 時區未標 | WARN（警告） |
| 用賭博網站作主要來源 | BLOCK（阻擋） |
