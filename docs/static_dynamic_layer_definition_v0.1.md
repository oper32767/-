# 靜態運算層與動態運算層定義 v0.1

日期：2026-06-17

## 1. 修正前錯誤

錯誤口徑：

```text
八字是靜態層，紫微是另一個靜態層，六壬奇門是動態層。
```

這樣講不精準，會導致八字、紫微各自被拆散，交叉驗證被降級成附屬步驟。

## 2. 修正後正式口徑

```text
靜態運算層 = 八字完整運算 + 紫微斗數完整運算 + 八字×紫微交叉驗證。
```

八字要完整跑：

1. 四柱。
2. 日主。
3. 月令。
4. 旺衰。
5. 格局。
6. 調候。
7. 用神／忌神／相神／病藥。
8. 十神結構。
9. 大運。
10. 流年。
11. 流月。
12. 吉凶月排序。

紫微斗數要完整跑：

1. 命盤。
2. 命宮／身宮。
3. 十二宮。
4. 主星／輔星／煞曜。
5. 廟旺陷。
6. 三方四正。
7. 四化。
8. 大限。
9. 流年。
10. 流月。
11. 新舊流月雙軌。
12. 事件場域與吉凶交叉。

八字與紫微都跑完後，才進入：

```text
八字×紫微交叉驗證。
```

這整包才叫靜態運算層。

---

## 3. 動態運算層正式口徑

```text
動態運算層 = 六壬完整起課／解課 + 奇門完整起局／解局 + 六壬×奇門事件交叉驗證。
```

六壬要完整跑：

1. 問題內容。
2. 起課時間。
3. 月將。
4. 天地盤。
5. 四課。
6. 三傳。
7. 課體。
8. 用神。
9. 類神／類將。
10. 神將類象。
11. 應期。
12. 事件裁斷。

奇門要完整跑：

1. 起局時間。
2. 陰遁／陽遁。
3. 局數。
4. 值符值使。
5. 九宮盤。
6. 天盤／地盤／門／星／神／暗干。
7. 空亡、馬星、門迫、入墓、擊刑等異常。
8. 用神。
9. 對衝宮。
10. 單宮整包。
11. 全局大象。
12. 應期與行動時窗。

六壬與奇門都跑完後，才進入：

```text
六壬×奇門事件交叉驗證。
```

這整包才叫動態運算層。

## 4. 新 Gate（檢核閘門）

| Gate（檢核閘門） | 判定 |
|---|---|
| 只跑八字就宣稱靜態層完成 | BLOCK（阻擋） |
| 只跑紫微就宣稱靜態層完成 | BLOCK（阻擋） |
| 八字、紫微未交叉就宣稱靜態層完成 | WARN（警告）或 BLOCK（阻擋） |
| 只跑六壬就宣稱動態層完成 | BLOCK（阻擋） |
| 只跑奇門就宣稱動態層完成 | BLOCK（阻擋） |
| 六壬、奇門未交叉就宣稱動態層完成 | WARN（警告）或 BLOCK（阻擋） |

## 5. 對第 61 串的接續

第 61 串紫微流月補丁應歸入靜態運算層內的「紫微斗數流月」與「八字×紫微月級交叉」部分。

不得歸入動態運算層。
