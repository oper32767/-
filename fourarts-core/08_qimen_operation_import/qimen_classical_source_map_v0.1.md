## 1. 主要來源

### 《遁甲演義》（四庫全書本）

可核公開版本：

- 維基文庫繁體版及卷一；
- 中國哲學書電子化計劃卷一；
- 必要時再對照原掃描影像。

此書可作為現代奇門排盤、解盤及部分運籌術語的上游來源之一，但不能跳過六十一串既有封板與雲龍裁決。

### 主客與運籌

原文包含：

- 五陽偏宜為客；
- 五陰宜為主；
- 九天宜揚兵；
- 九地宜潛藏；
- 太陰宜伏兵；
- 六合宜和合或避刑；
- 「急則從神，緩從門」；
- 天目為客、地為主；
- 三遁及其條件。

以上只是主題入口。每一條若要進規則庫，仍需：

1. 擷取完整上下文；
2. 對照至少兩個文本版本；
3. 標記異體字及轉錄疑點；
4. 與六十一串現行規格比對；
5. 建立條件、例外及優先序；
6. 經雲龍／冷審裁決。

## 3. 來源可信度分層

| 類型 | 用途 | 限制 |
|---|---|---|
| 原掃描影像 | 最終文字核對 | 閱讀成本高 |
| 維基文庫轉錄 | 快速搜尋與定位 | 可能有轉錄錯誤 |
| 中國哲學書電子化計劃 | 第二文本及英譯對照 | 同樣需核原圖 |
| 現代網站整理 | 搜尋線索 | 不作正式來源 |
| GitHub（程式碼託管平台）資料表 | 工程格式參考 | 不代表古籍來源正確 |

## 4. 禁止事項

- 不把古籍一句話拆掉前後文直接寫成萬用規則；
- 不把軍事語境直接等同現代商業或感情；
- 不因 GitHub 程式命中某格局，就認定現實事件必然發生；
- 不把同一古籍的多個轉載網站當成多票獨立證據；
- 不用模型補出古籍沒有寫的例外。
