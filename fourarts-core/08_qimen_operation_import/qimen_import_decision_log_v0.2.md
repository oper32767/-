- 標準化曆法快照；
- 多引擎統一輸出；

- 第一分歧差分工具；

- 公版古籍來源導航。

## 3. 驗收標準

合併前必須通過：

```bash
python3 tools/validate_qimen_import_package.py
python3 -m unittest discover -s tests -p "test_*.py" -v
```

自動檢查只證明工程結構正常，不裁決術數正誤。
